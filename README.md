# LinguaForge

An AI-powered language learning platform built with Next.js 14, featuring immersive exercises, spaced repetition, and gamification.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS
- **State:** Zustand
- **Animations:** Framer Motion
- **Validation:** Zod
- **Database:** Supabase (PostgreSQL)
- **3D/Graphics:** Three.js, React Three Fiber, Rive

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm run test` - Run tests
- `npm run quality` - Run quality checks

## Project Structure

```
src/
├── app/           # Next.js app router pages
├── components/    # React components
├── hooks/         # Custom React hooks
├── services/      # Business logic services
├── store/         # Zustand state management
├── schemas/       # Zod validation schemas
└── utils/         # Utility functions
```
