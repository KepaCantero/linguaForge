## Resumen Ejecutivo

El sistema actual tiene una base sÃ³lida basada en Krashen (comprehensible input), pero carece de:
- **SRS unificado** - No existe sistema de flashcards/repaso espaciado
- **Interactividad granular** - El texto no es clickeable por frase/palabra
- **Video integrado** - No hay reproductor de YouTube embebido
- **Contexto preservado** - Las frases se consumen sin metadata de origen

Este documento define la arquitectura para un sistema INPUT de clase mundial.
